람다식이란?

Step 0(가장 긴 코드)

Predicate<int> judge = 
    (int n) => {            // 이 부분이 람다식
        if(n % 2 == 0)      // 이 부분이 람다식
            return true;    // 이 부분이 람다식
        else                // 이 부분이 람다식
            return false;   // 이 부분이 람다식
    };                      // 이 부분이 람다식

var count = Count(numbers, judge);
  • 람다식은 일종의 메서드라고 생각하면 된다.
  • C# 2.0 시절에 있었던 delegate 키워드가 없어지고 =>(람다 연산자) 가 사용된다.
  • =>의 좌변에서 인수를 선언하고 오른쪽에는 메서드의 본문을 선언한다.
  • 이 람다식이 judge 변수에 대입되고 judge 변수가 Count 메서드의 인수에 전달된다.

var count = Count(numbers,
    (int n) => {
        if(n % 2 == 0)
            return true;
        else
            return false;
    }
);
  • judge 변수를 없애고 식을 직접 Count 메서드의 인수로 지정한다.
  • 변수의 형이 int 형이나 string 형일 때와 마찬가지이다.

Step 2

1

var count = Count(numbers, (int n) => { return n % 2 == 0; });
  • return의 오른쪽에 ‘n % 2 == 0’을 써 if 문을 없앨 수 있다.

Step 3

1

var count = Count(numbers, (int n) => n % 2 == 0);
  • 람다식에서 {}가 하나의 명령문을 포함할 때는 {}와 return을 생략할 수 있다.

Step 4

1

var count = Count(numbers (n) => n % 2 == 0);
  • 람다식에서는 인수의 형을 생략할 수 있다.

Step 5

1

var count = Count(numbers, n => n % 2 == 0);
  • 인수가 한 개인 경우에는 ()를 생략할 수 있다.
  • 이것이 람다식을 사용한 Count 메서드 호출의 최종적인 형태이다.
